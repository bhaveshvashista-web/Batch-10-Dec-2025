Insurance Charges Analysis

This project analyzes an insurance dataset to explore factors that influence medical insurance charges, such as age, gender, BMI, children, smoking habits, and region. Using Python and data visualization libraries, the project provides insights into how personal and lifestyle factors affect healthcare costs.
